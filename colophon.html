How This Site Is Built

This site is a static portfolio with a serverless contact form, designed to be fast, simple, and low-maintenance.

The frontend is built with plain HTML, CSS, and vanilla JavaScript. For a small, content-driven site, a framework wasn't necessary. Keeping the bundle small improves load time and reduces complexity.

The site is hosted on Cloudflare Pages and deployed directly from GitHub. Pages provides global CDN distribution and tight integration with Cloudflare Workers, which handle the contact form logic.

Form submissions are processed by a Cloudflare Worker running at the edge. The Worker verifies the request using Cloudflare Turnstile and sends the message via the Resend API. This avoids maintaining servers while still providing reliable email delivery with proper SPF and DKIM configuration.

Request Flow

  1. User submits the form
  2. Turnstile validates the request
  3. Worker processes the submission
  4. Resend sends the email

The entire flow completes in well under a second.

I documented this because infrastructure choices are part of system architecture. Choosing the simplest reliable solution matters more than over-engineering.
